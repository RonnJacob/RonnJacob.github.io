Nucleus

Android Application

Nucleus - A P2P Image Sharing App Employing Compression

April 2016

Nucleus is a peer-to-peer image sharing Android application that uses an optimal lossy image compression algorithm to reduce the size of images with minimal loss in the quality, and shares this image over a WiFi network using WiFi Direct technology. WiFi transfers are proven to be 50 times faster than Bluetooth and other short-wave radio communication techniques. By using a compressed image, Nucleus also helps the user to save storage space, a significant concern for user’s with limited memory. By using WiFi Direct, we take complete advantage of the WiFi medium, thus offering greater range and reduced transfer times. The functionalities provided by the WiFi direct technology were modified succesfully to make it a full duplex transfer medium.
